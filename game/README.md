# 怪兽农场游戏

这是一个怪兽农场经营模拟游戏，玩家可以培育作物、捕获怪兽、探索世界并发展自己的农场。

## 游戏特性

1. **作物耕种系统**
   - 多种作物类型，不同生长周期和产量
   - 怪兽辅助农田，提高生产效率

2. **怪兽捕获与养成**
   - 通过探索发现并捕获不同种类的怪兽
   - 不同稀有度的怪兽提供不同的能力加成

3. **资源管理**
   - 平衡金币、食物、材料和研究点等资源
   - 管理怪兽维护成本与收益

4. **科技树研发**
   - 解锁新的农场功能和怪兽能力
   - 提高资源生产效率和游戏进程

5. **探索系统**
   - 探索不同地区，获取资源和捕获怪兽
   - 随机事件与冒险任务

## 如何开始

1. 打开 `index.html` 或使用 `start_python_server.bat` 启动本地服务器
2. 按照游戏内教程开始你的怪兽农场之旅
3. 有关详细游戏指南，请参考 `/docs` 目录下的Wiki
