🌾 怪兽农场 Wiki 本地启动器

您的Wiki使用了Docsify框架构建，需要通过HTTP服务器来正确运行。以下是几种查看Wiki的方法：

方法1：使用Node.js的http-server（推荐）

如果您已安装Node.js，请按照以下步骤操作：

  1. 全局安装http-server：npm install -g http-server
  2. 进入wiki目录：cd 您的wiki目录路径
  3. 启动服务器：http-server
  4. 在浏览器中访问：http://localhost:8080

方法2：使用Python的简易HTTP服务器

如果您已安装Python，可以使用其内置的HTTP服务器：

Python 3.x

  1. 打开命令提示符，进入wiki目录
  2. 运行：python -m http.server 8000
  3. 在浏览器中访问：http://localhost:8000

Python 2.x

  1. 打开命令提示符，进入wiki目录
  2. 运行：python -m SimpleHTTPServer 8000
  3. 在浏览器中访问：http://localhost:8000

方法3：在线版本

您也可以直接访问在线版本的Wiki：

访问在线Wiki
