# 怪兽农场游戏文档

这里包含了怪兽农场游戏的详细文档和游戏指南，可以帮助玩家更好地理解游戏机制。

## Wiki目录

- **基础教程** (`basics.md`) - 游戏基本介绍和操作指南
- **怪兽图鉴** (`monsters.md`) - 所有可获得怪兽的详细信息
- **经济系统** (`economy.md`) - 游戏中的资源获取和管理
- **探索系统** (`exploration.md`) - 探索区域和捕获机制介绍
- **用户界面** (`ui.md`) - 游戏界面功能说明
- **存档系统** (`save.md`) - 游戏存档和云同步功能
- **开发文档** (`dev.md`) - 开发者文档和数据结构

## 浏览Wiki

在游戏目录下运行 `start_wiki_server.bat` 来启动本地文档服务器，然后在浏览器中访问文档。

或者直接打开 `index.html` 浏览目录。

## 内容更新

Wiki内容会随着游戏版本更新而更新，请确保查看与您当前游戏版本匹配的文档。
